# Note

## GrantNoteAction

Trigger this Action to add a Note to the players collected notes.

### Example

```json
{
  "type": "grantNote",
  "options": {
    "noteName": "<NOTE_NAME>"
  }
}
```

## ShowNoteAction

Trigger this Action to display a Note to the player.

### Example

```json
{
  "type": "showNote",
  "options": {
    "noteName": "<NOTE_NAME>"
  }
}
```
