# Inventory

## SelectedItemCondition

This Condition returns true if the player has the specified item selected in his inventory.

### Example

```json
{
  "type": "selectedItem",
  "options": {
    "itemName": "<ITEM_NAME>"
  }
}
```

## HasItemCondition

This Condition returns true if the player has the specified item in his inventory.

:::tip[Note]

### Example

```json
{
  "type": "hasItem",
  "options": {
    "itemName": "<ITEM_NAME>"
  }
}
```

## HasItemsCondition

This Condition returns true if the player has all the specified items in his inventory.

### Example

```json
{
  "type": "hasItems",
  "options": {
    "itemNames": [
      "<ITEM_NAME>",
      "<ITEM_NAME>"
    ]
  }
}
```
